Detailed Description

Main app management entry point responsible for the lifetime of the GUI's windows.

Note
Always ensure init() is called before everything else happens.
Only one instance of this can exist at any given time.
Windows can be created from any thread. The first window is always the main window and closing it will close all the other windows.
It is not advised to have an active shared_ptr handle to the window frame in the same scope as the run() command due to reference counting keeping the window alive even if the exit event was issued. You as the caller don't own anything the callee created.
